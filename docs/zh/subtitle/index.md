# 字幕设置

字幕设置用于分别调整原文字幕和译文字幕的显示样式。切换字幕类型后，下面的字体、颜色、背景、阴影、描边和实时预览都会作用于当前类型。

## 字幕类型

| 设置 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| 字幕类型 | 分段按钮 | 原文字幕 | 在原文字幕和译文字幕之间切换，分别编辑两套样式。 |

## 字体设置

| 设置 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| 字体系列 | 下拉框 | 原文：宋体；译文：Arial | 可选宋体、微软雅黑、Arial、Times New Roman、Helvetica。 |
| 字体大小 | 滑块 | 原文：18px；译文：16px | 范围 12px 到 48px。 |
| 预设文本 | 文本输入 | 原文：这是字幕预设文本；译文：This is translated subtitle preview | 实时预览中显示的示例文本。 |
| 字体颜色 | 颜色选择器 | 原文：`#ea9749`；译文：`#ffffff` | 支持颜色选择器和预设色按钮。 |
| 预设色 | 按钮组 | `#000000`、`#ff0000`、`#00ff00`、`#0000ff`、`#ffff00` | 快速套用常用字体颜色。 |
| 字体粗细 | 按钮组 | 400 | 可选 300、400、500、600、700、800。 |

## 显示设置

| 设置 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| 字幕背景 | 下拉框 | 半透明背景 | 可选无背景、半透明背景、纯色背景。 |
| 距底边距离 | 滑块 | 原文：80px；译文：120px | 范围 20px 到 200px，步进 10px。 |
| 背景色 | 颜色选择器 | `#000000` | 当背景为半透明或纯色时使用。 |
| 圆角 | 滑块 | 4px | 范围 0px 到 20px。 |
| 文字阴影 | 开关 | 关闭 | 开启后为字幕文字添加阴影。 |
| 文字描边 | 开关 | 关闭 | 开启后显示描边颜色和描边宽度设置。 |
| 描边颜色 | 颜色和文本输入 | `#000000` | 仅在文字描边开启时显示。 |
| 描边宽度 | 滑块 | 2px | 仅在文字描边开启时显示，范围 1px 到 5px。 |
| 实时预览 | 预览区域 | 当前样式 | 显示当前字幕类型的预览效果。 |

## 如何分别设置原文和译文字幕

1. 在字幕类型中选择原文字幕。
2. 调整字体和显示设置。
3. 切换到译文字幕。
4. 调整译文字幕样式并保存。刷新后播放器就会应用最新的字幕样式。

> 原文和译文字幕各自保存一套样式。切换字幕类型只改变正在编辑的目标，不会覆盖另一套设置。
