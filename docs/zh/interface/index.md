# 界面设置

界面设置控制 VideoMiner 前端显示语言和默认译文字幕语言。当前设置面板提供中文和英文两种语言选项。

## 语言设置

| 设置 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| 界面语言 | 下拉框 | 中文 (`zh`) | 控制前端界面文字。可选中文 (`zh`) 和 English (`en`)。 |
| 默认译文语言 | 下拉框 | 中文 (`zh`) | 视频播放和字幕编辑中默认展示的译文字幕语言。可选中文 (`zh`) 和 English (`en`)。 |

## 如何切换界面语言

1. 打开设置面板，进入界面设置。
2. 在界面语言中选择中文或 English。
3. 保存设置。

> 界面语言会与本地语言状态同步。默认译文语言只影响译文字幕的默认展示，不会自动重新翻译已有字幕。
