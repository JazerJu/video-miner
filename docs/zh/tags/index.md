# 标签管理

标签管理用于创建、编辑和删除视频标签。标签包含名称和颜色，列表支持多选后批量删除。

## 创建标签

| 设置 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| 标签名称 | 文本输入 | 空 | 新标签名称。按 Enter 或点击创建可提交。 |
| 标签颜色 | 颜色选择器 | 随机推荐色 | 新标签颜色，显示当前十六进制颜色值。 |
| 推荐颜色 | 按钮组 | 预设色列表 | 可选 `#3B82F6`、`#14B8A6`、`#10B981`、`#84CC16`、`#F59E0B`、`#F97316`、`#EF4444`、`#EC4899`、`#8B5CF6`、`#6366F1`、`#06B6D4`、`#64748B`。 |
| 随机 | 按钮 | 无 | 生成一个推荐的随机颜色。 |
| 创建 | 按钮 | 无 | 创建标签。标签名为空时不可点击。 |

## 标签列表

| 设置 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| 标签列表 | 列表 | 空 | 显示全部标签和标签数量。 |
| 选择标签 | 复选框 | 未选中 | 选择一个或多个标签用于批量删除。 |
| 编辑 | 按钮 | 无 | 进入编辑模式，可修改标签名称和颜色。 |
| 保存编辑 | 按钮 | 无 | 保存标签名称和颜色。 |
| 取消编辑 | 按钮 | 无 | 放弃当前编辑。 |
| 删除 | 按钮 | 无 | 删除单个标签，并从所有视频中移除该标签。 |
| 批量删除 | 按钮 | 无 | 删除选中的多个标签，并从所有视频中移除这些标签。 |
| 取消选择 | 按钮 | 无 | 清空当前选择。 |

## 如何编辑标签

1. 在"标签列表"中点击编辑。
2. 修改名称或颜色。
3. 点击保存。

## 如何删除标签

1. 删除单个标签时，点击该标签右侧的"删除"。
2. 批量删除时，勾选多个标签，再点击批量"删除"。
3. 在确认弹窗中确认操作。

> 当前设置面板没有”合并标签“控件。后端 API 存在合并能力，会在后续更新中加入。
