# Subtitle Settings

Subtitle Settings let you edit display styles separately for original subtitles and translated subtitles. After switching the subtitle type, the font, color, background, shadow, stroke, and preview controls apply to that selected type.

## Subtitle type

| Setting | Type | Default | Description |
| --- | --- | --- | --- |
| Subtitle Type | Segmented buttons | Original Subtitle | Switches between Original Subtitle and Translated Subtitle style editing. |

## Font settings

| Setting | Type | Default | Description |
| --- | --- | --- | --- |
| Font Family | Select | Original: SimSun; translated: Arial | Options are SimSun, Microsoft YaHei, Arial, Times New Roman, and Helvetica. |
| Font Size | Slider | Original: 18px; translated: 16px | Range is 12px to 48px. |
| Preview Text | Text input | Original: This is subtitle preview text; translated: This is translated subtitle preview | Text shown in the real-time preview. |
| Font Color | Color picker | Original: `#ea9749`; translated: `#ffffff` | Supports a color picker and preset color buttons. |
| Preset Colors | Button group | `#000000`, `#ff0000`, `#00ff00`, `#0000ff`, `#ffff00` | Quickly applies common font colors. |
| Font Weight | Button group | 400 | Options are 300, 400, 500, 600, 700, and 800. |

## Display settings

| Setting | Type | Default | Description |
| --- | --- | --- | --- |
| Subtitle Background | Select | Semi-transparent Background | Options are No Background, Semi-transparent Background, and Solid Background. |
| Bottom Distance | Slider | Original: 80px; translated: 120px | Range is 20px to 200px, step 10px. |
| Background Color | Color picker | `#000000` | Used when the background is semi-transparent or solid. |
| Border Radius | Slider | 4px | Range is 0px to 20px. |
| Text Shadow | Switch | Off | Adds shadow to subtitle text. |
| Text Stroke | Switch | Off | Shows stroke color and stroke width settings when enabled. |
| Stroke Color | Color and text input | `#000000` | Visible only when Text Stroke is enabled. |
| Stroke Width | Slider | 2px | Visible only when Text Stroke is enabled. Range is 1px to 5px. |
| Real-time Preview | Preview area | Current style | Shows the current subtitle style. |

## How to style original and translated subtitles

1. Select Original Subtitle.
2. Adjust font and display settings.
3. Switch to Translated Subtitle.
4. Adjust translated subtitle style and save. After a refresh, the player applies the latest subtitle styles.

> Original and translated subtitles keep separate style sets. Switching the subtitle type changes the target being edited and does not overwrite the other style set.
