# Interface Settings

Interface Settings control the frontend display language and the default translated subtitle language. The current settings panel provides Chinese and English options.

## Language settings

| Setting | Type | Default | Description |
| --- | --- | --- | --- |
| Interface Language | Select | Chinese (`zh`) | Controls frontend UI text. Options are Chinese (`zh`) and English (`en`). |
| Default Translation Language | Select | Chinese (`zh`) | Translation subtitle language shown by default in video playback and subtitle editing. Options are Chinese (`zh`) and English (`en`). |

## How to change the interface language

1. Open Settings, then Interface Settings.
2. Choose Chinese or English in Interface Language.
3. Save settings.

> Interface language syncs with the local language state. Default Translation Language changes the default display language for translated subtitles. It does not retranslate existing subtitles.
